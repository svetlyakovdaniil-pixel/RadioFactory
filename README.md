# RadioFactory

RadioFactory — новая система управления несколькими независимыми круглосуточными LIVE-станциями на одном или нескольких YouTube-каналах.

Текущий YouTubeFactory V1 остаётся рабочей системой и не изменяется до полной готовности RadioFactory.

## Статус

Проект находится на этапе архитектурного проектирования. Код новой системы ещё не пишется.
