# RadioFactory

RadioFactory — система управления несколькими независимыми круглосуточными LIVE-станциями на одном или нескольких YouTube-каналах.

Репозиторий пока используется для проектирования. Текущий YouTubeFactory V1 остаётся рабочим и не изменяется до полной готовности новой системы.
