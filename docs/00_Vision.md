# Vision

## Цель

Создать RadioFactory для управления несколькими независимыми LIVE-станциями на одном или нескольких YouTube-каналах.

## Принципы

1. Сначала полностью проектируем систему, затем пишем код.
2. Главный объект — Workspace.
3. Workspace содержит несколько YouTube-каналов.
4. Каждый YouTube-канал содержит несколько Station.
5. Каждая Station автономна и хранит собственный контент.
6. Общие механизмы находятся на уровне Workspace.
7. LIVE всегда важнее импорта и других фоновых задач.
8. Контент попадает в библиотеку только после успешной обработки.
9. Ошибки показываются понятно и конкретно.
10. Следующий LIVE запускается только после безопасного завершения и очистки предыдущего.
