# Vision

## Цель

Создать RadioFactory — систему управления несколькими независимыми круглосуточными LIVE-станциями на одном или нескольких YouTube-каналах.

Система должна позволять одному владельцу управлять радиосетью, в которой:

- один Workspace содержит несколько YouTube-каналов;
- каждый YouTube-канал содержит несколько Station;
- каждая Station ведёт собственный независимый LIVE;
- сбой одной Station не останавливает остальные;
- типовые проблемы устраняются автоматически;
- вся система видна в одном Dashboard.

## Основной сценарий

```text
Workspace
├── NoLyricsGroove
│   ├── Hip-Hop Station
│   ├── Lo-Fi Station
│   ├── Jazz Station
│   └── Chill Station
└── Cosmic Slumber
    ├── Deep Space Station
    └── Ambient Station
```

У каждой Station собственные музыка, loop-видео, превью, названия, описания, настройки LIVE, состояние, история и события.

## Граница первой версии

Первая версия проектируется сразу для нескольких YouTube-каналов и нескольких Station внутри каждого канала.

## YouTubeFactory V1

До полной готовности RadioFactory:

- V1 продолжает работать;
- V1 не перестраивается под новую архитектуру;
- RadioFactory создаётся отдельно;
- переход выполняется после тестирования;
- V1 архивируется, а не удаляется.

## Будущее

Архитектура не должна без необходимости связывать Station только с YouTube. Поддержка других платформ не является подтверждённой функцией первой версии.
