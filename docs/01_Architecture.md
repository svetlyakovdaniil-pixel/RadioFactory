# Architecture

```text
Workspace
├── YouTube Channel
│   ├── Station
│   └── Station
└── YouTube Channel
    ├── Station
    └── Station
```

## Workspace

Содержит YouTube-каналы, общие правила эфиров, Telegram, генератор метаданных, диспетчер ресурсов и общую панель мониторинга.

## YouTube Channel

Хранит:

- OAuth;
- настройки YouTube API;
- список плейлистов;
- доступ: всегда `Доступно всем`;
- аудитория: всегда `Не для детей`.

Не хранит регион, язык, часовой пояс и значения по умолчанию для новых Station.

RadioFactory показывает существующие плейлисты и автоматически создаёт новый плейлист по введённому названию, если его ещё нет.

## Dashboard

Главная страница показывает все YouTube-каналы, все Station, их статусы и быстрые действия запуска/остановки. Детальная работа выполняется внутри выбранной Station.
