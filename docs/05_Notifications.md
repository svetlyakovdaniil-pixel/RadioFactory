# Telegram Notifications

Telegram используется для оповещений, а не для полного логирования.

## Уровни

- критические ошибки;
- ошибки;
- предупреждения;
- информационные сообщения.

По умолчанию включены критические ошибки и ошибки.

## Антиспам

Одно событие создаёт одно итоговое уведомление. Промежуточные попытки восстановления отдельно не отправляются.

## Группировка

Если несколько Station пострадали от одной общей причины, приходит одно групповое сообщение со списком затронутых Station.
