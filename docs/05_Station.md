# Station

## Назначение

Station — автономная радиостанция внутри одного YouTube Channel.

## Собственные данные

Музыка, loop-видео, превью, названия, описания, длительность LIVE, состояние, история, события и подготовленный пакет следующего LIVE.

## Создание

Station создаётся через пошаговый мастер. Шаблоны и клонирование не используются. Шаги мастера ещё не детализированы.

## Управление

Station можно независимо запустить, штатно остановить, принудительно остановить, восстановить или перевести в Error.

### Остановить

Новые треки не запускаются, текущий доигрывает, LIVE корректно завершается, выполняется Cleanup.

### Остановить принудительно

После подтверждения `Остановить / Отмена` текущий трек может быть прерван, LIVE завершается немедленно.

## Waiting States

Отсутствие ресурса — ожидание, а не ошибка:

- Waiting for Music;
- Waiting for Loop Video;
- Waiting for Thumbnail;
- Waiting for Titles;
- Waiting for Descriptions.

Station продолжает подготовку автоматически после появления ресурсов.

## Error

Station переходит в Error после неудачного безопасного автоматического восстановления. Остальные Station продолжают работу.
