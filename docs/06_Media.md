# Media

## Библиотека

Каждая Station имеет отдельные музыку, loop-видео, превью, названия и описания.

## Импорт

- Drag & Drop;
- фоновые независимые очереди;
- видимый прогресс;
- отмена;
- понятные ошибки.

Файл попадает в готовую библиотеку только после успешной обработки и проверки.

## Управление

Поиск, сортировка, просмотр, переименование и удаление. Точные ограничения по типам ресурсов ещё не утверждены.

## Изменение музыки во время цикла

Текущий цикл не перестраивается. Следующий цикл формируется из актуальной библиотеки.

## Минимум для LIVE

- один обработанный трек;
- одно loop-видео;
- одно превью;
- одно название;
- одно описание.

Импорт не должен ухудшать стабильность LIVE.
