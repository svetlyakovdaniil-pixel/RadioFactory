# Broadcast

## Непрерывный режим

LIVE → естественное завершение → Cleanup → подготовка → следующий LIVE. Ожидание по расписанию между LIVE не используется.

## Запуск

Dashboard показывает запуск пошагово. Полный перечень этапов ещё не утверждён.

## Музыкальный цикл

Каждый цикл содержит каждый трек один раз в случайном порядке, без повторов. После завершения создаётся новый цикл. Каждый новый LIVE начинает новый цикл.

## Длительность

Следующий трек запускается только если полностью помещается в оставшееся время. Треки не обрезаются.

## Выбор ресурсов

Каждый LIVE случайно выбирает музыкальный цикл, loop-видео, превью, название и описание. При наличии альтернатив одинаковые loop-видео, превью, названия и описания не повторяются в двух последовательных LIVE.

## Pending LIVE Package

До создания LIVE формируется неизменяемый пакет ресурсов. При ошибке создания используется тот же пакет.

Новый пакет создаётся после успешного завершения предыдущего LIVE, отмены подготовки пользователем или инвалидирующего изменения ресурсов.

## Cleanup

Между LIVE выполняется изолированная очистка. Новый LIVE не запускается до её безопасного завершения. Состав Cleanup ещё не утверждён.
